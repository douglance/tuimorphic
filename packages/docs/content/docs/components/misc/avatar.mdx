---
title: Avatar
description: Display user images, initials, or icons for visual identification.
---

## Import

```tsx
import { Avatar } from 'tuimorphic';
```

## Basic Usage

<ComponentPreview>
  <AvatarBasic />
</ComponentPreview>

```tsx
<Avatar src="/user.jpg" alt="John Doe" />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| src | `string` | - | Image source URL |
| alt | `string` | `''` | Alt text for the image |
| initials | `string` | - | Initials to display when no image is provided or image fails to load |
| size | `'small' \| 'medium' \| 'large'` | `'medium'` | Size variant |
| href | `string` | - | Optional link URL - renders as anchor when provided |
| target | `string` | - | Link target (e.g., "_blank") |
| className | `string` | - | Additional CSS class names |
